최승아 포트폴리오 사이트 입니다

성실하게 꾸준히 나아가는 개발자 최승아 입니다

info personal project

부산경상대학교에 재학중인 최승아 입니다.

C#, JAVA, Javascript, PHP 등 해당 언어들로 여러 과제물을 만들며 실력을 쌓고 있고, 현재는 Vue.js를 배우고 있습니다.

해당 언어들로 여러 과제물을 제작하며 목표한 사이트를 만들기 위해 어설프지만 기획, 디자인, 프로그래밍 을 해봤습니다.

학과에 진학하고 초반에는 코딩을 해보기 전까지는 해당 직업이 개인적인 성향이 강한 업무라는 생각을 해왔습니다.

개발공부를 열심히 하면 되지 않을까 생각을 했지만 2학년이 되고 여러 결과물을 만들어내며 여러 사람들과 소통을 하는 일이 많았고

코드를 잘 쓰는 것도 중요하지만 주변 사람들과 여러 의견을 공유하며 각종 피드백과 조언 등을 받아들이고 수정해나가며

원활한 커뮤니케이션과 컨디션을 유지하는것도 중요하다 느꼈습니다.

개발에 필요한 공부를 성실히 하며, 겸손한 태도로 꾸준히 나아가는 모습 보여드리겠습니다.
